# A股场外基金交易估值账本

这是一个本地运行的小 Web 应用，用 SQLite 保存你的基金、交易流水和每日估值。

## 启动

```powershell
cd C:\Users\Administrator\Documents\Codex\2026-07-02\a\work\fund_dashboard
python app.py
```

电脑浏览器打开：

```text
http://127.0.0.1:8765
```

iPhone 和电脑在同一个 Wi-Fi 下时，手机 Safari 打开电脑的局域网地址，例如：

```text
http://192.168.31.144:8765
```

然后在 Safari 分享菜单里选择“添加到主屏幕”，即可像 App 一样打开。

## 日常使用

1. 添加基金代码和名称。
2. 第一次使用时，先在“录入历史持仓”里填基金代码和已有份额。
3. 点击“刷新估值”，或保持应用开着等待后台每 10 分钟自动刷新。
4. 以后每天只记录新增买入、赎回、分红等交易。
5. 白天看“今日估算盈亏”，晚上正式净值出来后，应用会用正式净值校准。

## 数据来源

- 基金名称和盘中估值：优先使用天天基金的盘中估算接口。
- 正式净值：使用东方财富基金净值接口。
- 盘中估值不是基金公司官方正式净值，只适合收盘前参考；正式复盘以晚间/次日披露的正式净值为准。

## 盈亏口径

- 当前份额 = 历史持仓份额 + 持仓日期之后的交易份额变化。
- 今日估算盈亏 = 昨日已持有份额 * (今日最新净值 - 上一交易日净值)。
- 当天新买入的场外基金通常按当天收盘净值确认，因此不计入当天老持仓盈亏。
