# A股场外基金交易估值账本

这是一个本地运行的小 Web 应用，用 SQLite 保存基金份额、每日估算和已披露净值。

## 启动

```powershell
cd C:\Users\Administrator\Documents\Codex\2026-07-02\a\work\fund_dashboard
python app.py
```

电脑浏览器打开：

```text
http://127.0.0.1:8765
```

iPhone 和电脑在同一个 Wi-Fi 下时，手机 Safari 打开电脑的局域网地址，例如：

```text
http://192.168.31.144:8765
```

然后在 Safari 分享菜单里选择“添加到主屏幕”，即可像 App 一样打开。

## 日常使用

1. 第一次使用时，通过“改份额”录入上一个交易日确认的基金份额。
2. 份额发生变化后再次使用“改份额”；每次修改都会保留历史版本。
3. 交易日上午和下午每 3 分钟自动记录一次盘中估算，收盘后每 10 分钟检查已披露净值。
4. 白天看预估盈亏，晚上结合“净值更新”进度查看已确认盈亏。

## 数据来源

- 基金名称和盘中估算：天天基金接口，属于第三方估算，不是基金公司官方数据。
- 已披露净值：东方财富基金净值接口，属于第三方转发的基金已披露净值，不是基金公司直连接口。
- 页面始终同时显示数据日期；接口失败或数据日期不是今天时，不会把旧值冒充为今日实际值。

## 盈亏口径

- 当前份额 = 最近一次保存的已确认份额。
- 今日预估盈亏 = 上一交易日已确认份额 × (今日盘中估算净值 - 上一交易日净值)。
- 已确认盈亏只汇总已披露今日净值的基金，并同时显示确认覆盖数量，避免与全组合预估混淆。
